Git 的配置

Git 的三个配置文件

Git 自带一个 git config 工具来控制 Git 的外观和行为

  • /etc/gitconfig文件：用 sudo git config --system 进行配置，针对整个系统
  • ~/.gitconfig或~/.config/git/config文件：用 git config --global 配置，针对当前用户
  • .git/config 文件：用 git config --local 配置，针对某一工作目录

每一个级别会覆盖上一级别的配置
